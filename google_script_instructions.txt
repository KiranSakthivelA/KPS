# Google Sheets Integration Guide
To push your website booking data to a Google Sheet automatically, follow these steps:

1. Create a new Google Sheet at https://sheets.google.com
2. Add these headers to the first row (A1 to G1):
   Name | Phone | Pickup | Drop | Car Type | Travel Date | Message
3. Click on "Extensions" -> "Apps Script" in the top menu.
4. Replace all code in the editor with the code below:

```javascript
function doPost(e) {
  try {
    const sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
    const data = JSON.parse(e.postData.contents);
    
    // Add row to sheet
    sheet.appendRow([
      data.name || '',
      data.phone || '',
      data.pickup || '',
      data.drop || '',
      data.car || '',
      data.date || '',
      data.message || '',
      new Date() // Timestamp
    ]);
    
    return ContentService.createTextOutput(JSON.stringify({"status": "success"}))
      .setMimeType(ContentService.MimeType.JSON);
  } catch(error) {
    return ContentService.createTextOutput(JSON.stringify({"status": "error", "message": error.toString()}))
      .setMimeType(ContentService.MimeType.JSON);
  }
}
```

5. Click the "Save" icon (disk).
6. Click the "Deploy" button at the top right -> "New deployment".
7. Select type: "Web app" (gear icon next to "Select type").
8. Set "Execute as" to "Me" and "Who has access" to "Anyone".
9. Click "Deploy" (you may need to authorize the script).
10. Copy the "Web app URL" provided.
11. Open `api/submit_inquiry.php` and replace 'YOUR_GOOGLE_SCRIPT_WEB_APP_URL' with the URL you just copied.
